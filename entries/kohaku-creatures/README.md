# kohaku-creatures

Companion creature templates focused on conversational + roleplay
personalities, separate from kt-biome's task-oriented bundle.

These creatures lean on persona-shaped system prompts more than
tool surfaces — the goal is rich character behaviour over getting
work done.  Mix and match with `general` from kt-biome for a
character that can also execute tasks.

## Install

```bash
kt install @kohaku-creatures
```
