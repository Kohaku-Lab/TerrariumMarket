# kohaku-extensions

Drop-in tool and plugin extensions to expand what creatures can do.

Contents:

| Kind | Examples |
|---|---|
| Tools | Extra web search providers, file format converters, integrations |
| Plugins | Cross-cutting behaviour add-ons (cost tracking, audit log shapes) |

## Install

```bash
kt install @kohaku-extensions
```

After install, the new tools become available to any creature that
opts them into its `tools:` list, and plugins activate per
creature config.
